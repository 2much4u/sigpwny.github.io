SIGPwny

    University of Illinois at Urbana-Champaign

  sigpony-l-owner@acm.uiuc.edu

SIGPwny is a student-run information security interest group at UIUC. We host weekly public meetings on recent security news and low-barrier-of-entry security topics. In addition, we coordinate weekly focus groups for more in-depth exploration in topics like embedded hacking and penetration testing.

SIGPwny is spelled SIGPwny, but some alums might remember it as SIGPony.

SIGPwny is mentored by Kirill Levchenko and Joshua Mason.
